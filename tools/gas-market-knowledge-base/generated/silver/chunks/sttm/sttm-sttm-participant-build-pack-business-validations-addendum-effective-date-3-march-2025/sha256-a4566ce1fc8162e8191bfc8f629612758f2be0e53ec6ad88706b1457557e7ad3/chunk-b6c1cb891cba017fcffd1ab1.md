The allocation agent must submit Preliminary, Final and Revision Facility allocations to AEMO for market settlement for each billing period.

Table 23 Facility Allocation Interface Format

gasdate, M/O/NR = M. gasdate, Description = The date of gas day. crn, M/O/NR = M. crn, Description = The unique identifier of the Registered Facility Service. Refer to report INT720 - Facility Operator Registered Services for a complete list of Registered Services. facilityid, M/O/NR = M. facilityid, Description = The STTM unique identifier for the facility. Refer to report INT671 - Hub and Facility Definitions for a complete list of all STTM facilities.. allocationquantity, M/O/NR = M. allocationquantity, Description = The quantity of the net flow allocated to the Registered Facility Service which takes into account of any positive MOS quantity or negative MOS quantity allocated to the Registered Facility Service.
