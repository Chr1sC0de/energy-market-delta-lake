Table 18 Contingency Gas Bid/Offer Process Validations

Flow Direction validation, Event Code = 4704. Flow Direction validation, Description = The Contingency Gas Bid/Offer must be associated with a valid directioncode.. Price\Quantity validation, Event Code = 4706. Price\Quantity validation, Description = The first Price\Quantity step must have non-null values.. Contiguous Steps, Event Code = 4707. Contiguous Steps, Description = Step Price\Quantity values must be contiguous.. Suspended STTM User or Shipper, Event Code = 4717, 4718. Suspended STTM User or Shipper, Description = A Trading Participant who is suspended at a hub in the role of STTM User must not bid for Contingency Gas as an STTM User withdrawing gas at that Hub. A Trading Participant who is suspended at a hub in the role of STTM Shipper must not bid for Contingency Gas on an STTM Facility shipping gas to or from the that Hub.. Single record per file, Event Code = 4008. Single record per file, Description = Only one CG bid or offer is allowed per file.
