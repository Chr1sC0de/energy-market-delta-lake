Table 30 Registered Facility Service Allocation Process Validations
Data Provider, Event Code = 4600. Data Provider, Description = This error code indicates that the Registered Facility Service Allocation has not been provided by the organisation nominated as the Allocation Agent for the Registered Facility Service in the STTM.. Mandatory Fields, Event Code = 4611. Mandatory Fields, Description = All mandatory fields for the allocation files must be provided.. Inactive Registered Facility Service, Event Code = 4602. Inactive Registered Facility Service, Description = The Registered Facility Service Allocation file must be associated with a Registered Facility Service that is active for the gas date for which the allocation data is submitted.. Missing Trading Rights, Event Code = 4603. Missing Trading Rights, Description = The Registered Facility Service Allocation file must contain allocation data for every associated trading right.. Duplicate Allocation Record For Gas Day, Event Code = 4631. Duplicate Allocation Record For Gas Day, Description = Registered Facility Service Allocation file must not contain duplicate records for a gas day.. Invalid Trading Rights, Event Code = 4637. Invalid Trading Rights, Description = Registered Facility Service Allocation file must contain allocations for all active Trading Rights for the Registered Facility Service.. Invalid Hub, Event Code = 4639. Invalid Hub, Description = The STTM Hub
