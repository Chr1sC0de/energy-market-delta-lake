Trigger, 1 = This transaction is created when the Allocation Agent has determined the MOS allocations for a MOS stack for a gas day.. Rules and Procedures References, 1 = Rules 419, 421. Pre-condition, 1 = The latest published MOS stacks for each gas day covered by the allocations must be used to determine the MOS step allocations. The key market system data source for this transaction is MIS Report 721.. Post-condition, 1 = The MOS step allocation file is validated by market systems and used in Settlement calculations.. Variations, 1 = This file may be sent with the following file descriptor types: 'MSD' - Daily File 'MSU' - Update File 'MSP' - Preliminary File 'MSF' - Final File 'MSR' - Revision File Daily allocations must be sent by Gas Day Start + 4:30 Hours, the day after the gas day to which the allocations relate.
