SWEXIE (SWEX Interface Engine) allows the upload of files to the STTM database via an FTP to the market systems or CSV file upload via SWEX, and is a simplified process of submitting bids, offers, withdrawals, allocations, variations and transactions.
Details of the form of CSV transaction are described in the STTM Participant Buildpack and the STTM Participant Buildpack Business Validation Addendum, which are part of the STTM Interface Protocol (SIP).
