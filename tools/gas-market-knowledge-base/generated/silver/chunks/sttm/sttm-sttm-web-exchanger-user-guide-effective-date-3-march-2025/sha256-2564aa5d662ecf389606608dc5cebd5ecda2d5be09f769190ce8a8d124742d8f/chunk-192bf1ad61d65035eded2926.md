To add an allocation agent to a trading right, select the trading right
1.  On the Home page, click Registered Services & Trading Rights .
2.  On the Registered Services drop-down menu, select the relevant contract.
3.  Click Retrieve Registered Service.
4.  Click the Trading Rights icon . This is the coloured icon to the right of the Registered Service Details section.
This brings up the dialog boxes for inserting another trading right ( Edit / Insert Trading Right ) as well as the dialog box for nominating the allocation agent ( Registered Service Allocation Agent Details ).
