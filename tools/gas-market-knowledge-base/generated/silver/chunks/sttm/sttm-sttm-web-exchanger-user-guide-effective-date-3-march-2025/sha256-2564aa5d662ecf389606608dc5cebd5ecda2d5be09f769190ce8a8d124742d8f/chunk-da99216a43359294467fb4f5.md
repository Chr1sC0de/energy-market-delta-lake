This section describes how CSV files can be manually uploaded directly to SWEXIE and the STTM database from a SWEX screen. This upload functionality can be used as an alternative to accessing SWEXIE via the FTP server.
For information on the format of CSV files, see the STTM Participant Build Pack for more information.
