10.5.4(a).. HP(d), Definition = The ex ante market price for a hub for gas day d. This value is determined in accordance with rule 417.. IHP(d), Definition = The ex post imbalance price for a hub for gas day d. This value is determined in accordance with rule 426.. LD(d,k), Definition = A value of 0 or 1 to control whether or not a Trading Participant's deviation for gas day d which decreases net supply (i.e. supply less consumption) to a hub is included in the allocation of settlement surpluses and shortfalls for the billing period containing gas day d. The daily deviation is included if LD(d,k) = 1, but not if LD(d,k) = 0. The value of LD(d,k) is determined in clause 10.10.2.. LDQ(p,d), Definition = The long deviation quantity for Trading Participant p at a hub on gas day d..
