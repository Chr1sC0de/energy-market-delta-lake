AEMO shall send a message acknowledgement for every message it receives. The Message Identifier component of the file name will be filled in with the Message Id of the message submitted. This would facilitate better traceability between the submitted message and corresponding acknowledgement (InitiatingMessageId). In the unlikely event of a file being submitted with a blank or invalid Message Identifier component, AEMO ReceiptId will be used instead.
The information in the file describes whether or not the file submitted conforms to specifications.
