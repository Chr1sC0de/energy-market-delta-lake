Connections to all externally facing applications will use 2-factor authentication; one to the network and the other to the application. Existing AEMO access mechanisms are proposed for the STTM. This provides:
- Network VPN from participant networks
- Client VPN from participant workstations
- Client access via browser using HTTPS (via VPN)
Once connected to the network, user name and password will be required to authenticate against the application domain.
