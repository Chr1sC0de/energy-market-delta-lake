This report contains information on participant's daily long/short gas positions and is used by participants to identify a counterparty for off-market MSV trades.
Note: This report does not have a unique key. The Primary Key combination specified in the table below should be used as a guide only.
Access:
TP
Issued By:
When the auto deviation calculation job is run successfully at 11.38 every day. It is also issued when the prudential monitoring job is run successfully on business days.
Report Period:
Gas days greater than or equal to report date minus SEVEN days
Trigger:
Successful completion of the auto deviation calculation job and prudential run

int724_v1_ranked_deviation_quantities_rpt_[pid]~yyyymmddhhmmss
