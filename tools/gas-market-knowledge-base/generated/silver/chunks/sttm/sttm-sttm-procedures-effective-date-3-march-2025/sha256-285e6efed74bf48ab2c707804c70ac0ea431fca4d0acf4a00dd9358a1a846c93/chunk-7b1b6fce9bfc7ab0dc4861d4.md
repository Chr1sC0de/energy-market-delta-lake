The following table defines the sets used in the settlement equations.
AH, Definition = Denotes the set of registered trading rights for as available capacity that allow the supply of gas to a hub . This set does not include registered trading rights that allow withdrawal of gas from the hub .. BP, Definition = Denotes the set of gas days in a billing period .
FH, Definition = Denotes the set of registered trading rights for firm haulage that allow the supply of gas to a hub . This set does not include registered trading rights that allow withdrawal of gas from the hub .. SN, Definition = Denotes the set containing a single member representing all of the STTM distribution systems for a hub .. SP, Definition = Denotes the set of STTM facilities serving a hub .
