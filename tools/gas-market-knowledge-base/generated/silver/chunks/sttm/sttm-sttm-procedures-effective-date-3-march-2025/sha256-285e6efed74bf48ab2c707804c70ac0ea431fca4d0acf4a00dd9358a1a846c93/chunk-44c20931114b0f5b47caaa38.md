- (a) If AEMO receives an updated allocation notice under clause 7.2.3 it must substitute the values provided in that allocation notice for those provided in the original allocation notice under rule 419(1).
- (b) AEMO must retain the values provided in an earlier allocation notice if it:
- (i) does not receive an updated allocation notice under clause 7.2.3; or
- (ii) the updated allocation notice is rejected in accordance with rule 419(3).
