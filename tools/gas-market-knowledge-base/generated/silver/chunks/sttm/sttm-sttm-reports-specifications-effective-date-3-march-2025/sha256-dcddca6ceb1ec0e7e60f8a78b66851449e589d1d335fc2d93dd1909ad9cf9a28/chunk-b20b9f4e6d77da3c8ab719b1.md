This report contains information on STTM users withdrawing gas on Deemed STTM Distribution systems. Note that this report includes information that may be used in determining the Transmission Connected STTM User Allocation submissions.
This report is made available to pipeline operators on hubs with deemed STTM distribution systems.
Access :
Pipeline Operator
Issued By :
4 PM Daily
Report Period :
All Transmission Connected STTM Users holding capacities at each Deemed STTM Distribution system where the period end date of the capacity is greater than or equal to today minus 31 days.
