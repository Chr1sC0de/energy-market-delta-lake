This report contains MOS stack data for an STTM pipeline facility which is valid for the gas day commencing on the day before the report is published. It contains quantity and physical contract identifier(s) for each MOS stack step and pricing data.
The data contained in this report is published on gas day D+1 and identifies all MOS stack data which is valid for gas day D. This report in conjunction with the INT720A report that is published on gas day D-1 represents the data that will be used by AEMO to validate the MOS Allocation files submitted on gas day D+1. Facility Operators may use this data to verify that their MOS Allocation files contain valid data in order to minimise the risk of the files being rejected.
