This report contains information on all the Registered Service records for a facility operator. Note that this report includes information on active Registered Services and may be used in determining the Facility Allocation submissions.
Also note that the primary key definition for this report will not hold if the contract issuer has rejected (or confirmed) 2 or more capacities for the registered service covering the exact same date range in one submission via SWEX.
This report is made available to both network operators and to pipeline operators.
