This report contains individual trading participant's deviation and variation data.
Note that the primary key definition for the report may not always hold. The report can contain multiple entries of the same participant of the same charge method and charge payment type for the same gas day for different flow directions.
Access:
TP
Issued By:
When the auto deviation calculation job is run successfully at 11:38 When prudential is run successfully
Report Period:
Gas days greater than or equal to report date minus SEVEN days
All gas days and facilities for which deviation data has been updated in the seven days prior to the report date
Trigger:
Successful completion of the auto deviation calculation job and prudential run
