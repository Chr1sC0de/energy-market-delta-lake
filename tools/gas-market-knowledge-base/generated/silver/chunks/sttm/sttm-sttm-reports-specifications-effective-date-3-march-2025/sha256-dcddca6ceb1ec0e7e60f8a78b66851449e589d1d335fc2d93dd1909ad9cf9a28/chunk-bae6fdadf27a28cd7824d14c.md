This report contains information on all the Registered Service records for a facility operator. This report provides the same data as INT720 except for the inclusion of the two additional fields: contract_holder_identifier and contract_holder_name. Note that this report includes information on active Registered Services and may be used in determining the Facility Allocation submissions.
Also note that the primary key definition for this report will not hold if the contract issuer has rejected (or confirmed) 2 or more capacities for the registered service covering the exact same date range in one submission via SWEX.
This report is made available to both network operators and pipeline operators.

All Registered Service records where the crn_end_date is greater than or equal to today minus 31 days.
