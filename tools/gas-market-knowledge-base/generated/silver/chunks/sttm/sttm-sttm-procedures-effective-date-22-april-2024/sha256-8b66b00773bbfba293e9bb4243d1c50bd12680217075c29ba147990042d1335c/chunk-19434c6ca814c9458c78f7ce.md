- (a) For Trading Participant p with a Iong deviation quantity for withdrawals from the hub (i.e. lower withdrawal than expected) on market facility k on gas day d, the deviation price is:
If DPFlag(d) = 0
```
If CGPH(d) ≥ 0 PDevPF(p,d,k) = MAX( MINP(d), MIN( MAXP(d), HP(d), IHP(d), CGPL(d) )) Else PDevPF(p,d,k) = MAX( MINP(d), MIN( MAXP(d), HP(d), IHP(d), CGPL(d), MOSXD(d) )) If DPFlag(d) = 1 PDevPF(p,d,k) = HP(d)
```
