1, Description = Facility operator updates contract information Facility operator informs AEMO about changes to contracts they have with shippers.. 2, Description = Shipper selects contract Participant logs in to AEMO Markets Portal and selects the contracts they wish to be adjusted by a successful trade. A contract is selected for each product they wish to bid for from a list of valid contract references for the facility supplied by the facility operator.. 3, Description = Shipper submits orders to the platform (Trayport) AEMO validates that the participant has pre-selected a contract (includes pre-matched trades).. 4, Description = Trades are executed on the CTP Bids are matched with offers to form a trade, and traders receive confirmation (includes pre-matched trades).. 5, Description = AEMO performs delivery netting AEMO provides the facility operator with the total net capacity each shipper has sold or purchased. The netting process for forward-traded products occurs 14 days prior to the gas day. The netting run looks at each zone-to-zone (or park) product that has been traded for each gas day in the next 14-day delivery window.. 6, Description = AEMO sends capacity transfer notice to the facility operator AEMO applies contract and point information
