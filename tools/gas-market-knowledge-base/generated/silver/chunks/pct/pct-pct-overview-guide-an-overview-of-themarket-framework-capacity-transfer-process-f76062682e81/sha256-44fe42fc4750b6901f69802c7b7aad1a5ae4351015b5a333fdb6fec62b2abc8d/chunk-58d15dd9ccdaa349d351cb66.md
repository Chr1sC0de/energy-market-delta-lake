The OTS Code is a new instrument that governs the content of standard operational transportation service agreements (OTSA). The code contains descriptions of each standard OTS, the standard terms and conditions for the provision and use of a standard OTS, requirements for facility-specific terms, and a form of agreement for execution by the parties.
