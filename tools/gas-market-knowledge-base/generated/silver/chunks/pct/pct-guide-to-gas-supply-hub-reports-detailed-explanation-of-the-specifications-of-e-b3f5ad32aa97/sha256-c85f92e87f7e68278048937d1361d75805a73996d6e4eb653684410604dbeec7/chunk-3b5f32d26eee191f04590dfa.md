The general format of the report filename is:
For example:
Table 1 below details the format of each component in the filename.
Table 1 Report filename components

[PARTICIPANT_CODE], Description = The participant code for private participant reports, or the word PUBLIC for public reports. [PARTICIPANT_CODE], Format = Four alphanumeric characters, or 'PUBLIC'. [FILE_ID], Description = The report name. [FILE_ID], Format = Up to 28 alpha characters long. [PERIOD_ID], Description = The report period identification. [PERIOD_ID], Format = YYYYMMDD. [CONTENT_ID], Description = The file content id, this is an optional part of the filename, it only applies to some reports. [CONTENT_ID], Format = Up to 20 alphanumeric characters long. [EVENT_QUEUE_ID], Description = The unique identifier of the file. [EVENT_QUEUE_ID], Format = Greater than or equal to 1 numeric character long and less than or equal to 30 numeric characters long.
