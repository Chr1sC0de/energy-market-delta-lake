-  This report contains fourteen report subtypes:
- -Settlement run: provides header details of the settlement run. Note that initial settlement run covers the period from the first gas date of the billing period for which a final statement has not been issued until yesterday's gas date. Adjustment settlement run covers the period for which a final settlement has been run but a revision settlement has not been run.
- -Settlement summary: provides summary details of the settlement run.
