For each day, a participant submits a single bid profile, which contains all bids (including bid steps) for all products on that day. Because only one bid profile can be submitted for each day, any new bid profile submission completely overwrites any previous submission for that day. In other words, if a participant wants to edit any bid included in the profile, they must resubmit a full profile including the edited bid and any other unedited bids for that day.
Participants can view their active bid profiles through the interface or extract them via an API call.
