As shown in Figure 3, the GRMS is comprised of three separate gateways:
- webMethods Gateways (EbXML) -the gateway to the FRC Hub that is used to send and receive ebXML messages;
- FTP Server (CSV) -allows participants and the GRMS to exchange messages in nonaseXML/ebXML format (i.e. CSV format files); and
- SMTP Server (e-mail) -stores all incoming and outgoing email messages.
The GRMS Transactions Manager database provides the integration between the webMethods Gateway, the FTP Server and the SMTP Server and is used to store all incoming and outgoing messages and transactions.
Figures 4 and 5 show two gateway examples relating to aseXML messages routed via the WebMethods Gateway, and to messages routed via the FTP Server.
