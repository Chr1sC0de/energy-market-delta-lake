The /Meter Installation Registration Number MIRN) is the term used to describe a unique gas metering point.  The MIRN Discovery transaction group is used to return MIRN Standing Data information  given  the  address  of  a  MIRN,  or  the  MIRN  itself.    The  transaction  is  conducted between Users and Network Operators.
