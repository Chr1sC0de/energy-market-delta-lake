The Missing Energy Data transaction is used by a User to request any energy data that has not been received from a Network Operator by the expected date.  The Network Operator will obtain the requested data and provide it to the User via the Meter Data Notification transaction. This may be either a special transaction in response to this request or the next scheduled transaction.
