Trigger, 1 = Network Operator receives a transfer confirmation from AEMO. Pre-conditions, 1 = Customer transfer request has been fully approved. Transaction acknowledgment specific event codes, 1 = 3669 (Also the generic event codes 3603, 3659, 3662, 3673 can be used)
