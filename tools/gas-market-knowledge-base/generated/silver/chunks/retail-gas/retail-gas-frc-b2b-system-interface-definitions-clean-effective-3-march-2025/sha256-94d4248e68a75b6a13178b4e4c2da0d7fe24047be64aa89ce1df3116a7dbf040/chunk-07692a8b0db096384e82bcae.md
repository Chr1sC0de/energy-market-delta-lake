A  User  will  obtain  the  Meter  Index  Data  on  disconnection  of  a  meter.    The  User  uses  a MeterReadInputNotification transaction to send the index data to the Network Operator for storage and later use for energy flow calculations.
The diagram below shows the sequence of events for this transaction:
Figure 4-10 Meter Read Input Sequence Diagram (SA only)
1, AseXML Transaction = MeterReadInputNotification. 1, From Object = User. 1, To Object = FRC Hub. 1, Process Flow = MR9B. 2, AseXML Transaction = MeterReadInputNotification. 2, From Object = FRC Hub. 2, To Object = Network Operator. 2, Process Flow = MR9B
