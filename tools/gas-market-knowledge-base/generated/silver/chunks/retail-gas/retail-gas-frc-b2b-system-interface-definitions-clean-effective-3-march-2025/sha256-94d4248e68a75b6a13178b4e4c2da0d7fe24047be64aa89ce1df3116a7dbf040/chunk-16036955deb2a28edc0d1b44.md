Special Read Request logged by Network Operator 2. Network Operator has closed Work Request. Post-conditions, 1 = 1. User has a Work Request number from the Network Operator 2. User closes Special Read Request. Transaction acknowledgment specific event codes, 1 = 3602 (Also the generic event codes 3603, 3659, 3662, 3673 can be used)
