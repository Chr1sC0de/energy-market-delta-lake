This document focuses on the specific aseXML interfaces to be used in the B2B transactions. The interfaces focus on business and application logic. The interfaces are grouped based on the transaction categorisation in the Consolidated Transaction List.
Every group of specific interfaces has one or more activity diagrams associated with it. These activity diagrams are based on the B2B Process Flow diagrams.
Each sequence diagram has a table associated with it. Each row in this table describes correlation between the process flow and the corresponding aseXML transaction.
