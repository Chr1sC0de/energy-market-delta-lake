Transaction Definition Table cross-reference, 1 = This interface realises the following transactions from the GPTWG Transaction Definition Table: • 281 - MIRN Standing Data • 284 - MIRN Additional Data Note. The NMIDiscoveryResponse transaction also realises these transactions when the request is a NMIDiscoveryRequest transaction.. Trigger, 1 = Completion of processing of the NMIStandingDataRequest transaction.. Pre-conditions, 1 = None. Post-conditions, 1 = User has the MIRN Standing Data, or a record of failure of processing of the NMIStandingDataRequest data and all errors detected.
Transaction acknowledgment specific event codes, 1 = 3602 (Also the generic event codes 3603, 3659, 3662, 3673 can be used)
The NMIStandingDataResponse transaction provides the MIRN Standing Data to the requestor or advises of the failure of the processing of the issued NMIStandingDataRequest transaction.  It also identifies whether any additional NMI Standing Data will be issued at a later time to the User in a subsequent NMIStandingDataResponse.
