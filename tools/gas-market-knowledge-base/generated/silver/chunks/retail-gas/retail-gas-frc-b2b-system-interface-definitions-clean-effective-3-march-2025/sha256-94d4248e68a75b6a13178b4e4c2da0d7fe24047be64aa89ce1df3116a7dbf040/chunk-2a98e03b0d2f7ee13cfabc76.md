- (a) If a Retailer disagrees with the consumption, notification will not be via the Dispute process but via the MDV process.  In which case, if a energy reading is revised for a particular day / line item,  then  we  understand  that  all  reads  thereafter  will  need  to  be  unaccumulated  and  reaccumulated to the end of the billing period.   This will generate the required Cancel/Rebill lines.
