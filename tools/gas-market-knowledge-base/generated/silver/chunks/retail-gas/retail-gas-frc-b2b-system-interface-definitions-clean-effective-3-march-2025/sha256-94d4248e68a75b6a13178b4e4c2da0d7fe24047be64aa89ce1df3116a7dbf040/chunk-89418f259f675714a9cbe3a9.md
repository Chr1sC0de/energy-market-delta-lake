Dispute_Reason_Code, 1 = Dispute Reason Code. Dispute_Reason_Code, 2 = Applicable Dispute Reason Code. Dispute_Reason_Code, 3 = String. Dispute_Reason_Code, 4 = 4. Dispute_Reason_Code, 5 = "NNMI" = MIRN not known to User (i.e. User does not supply customer). "BPDF" = Billing Period Different 'FC1' = Fixed Charge item 1 different (WA only) 'FC2' = Fixed Charge item 2 different (WA only) 'FC3' = Fixed Charge item 3 different (WA only) 'FC4' = Fixed Charge item 4 different (WA only) 'VC1' = Variable Charge item 1 different (WA only) 'VC2' = Variable Charge item 2 different (WA only) 'VC3' = Variable Charge item 3 different (WA only) 'VC4' = Variable Charge item 4 different (WA only) "LRTB" = User has lost customer to another User before the invoiced period. "LRTD" = User lost customer to another User during the invoiced period (therefore need to apportion network charge between old and new User).
