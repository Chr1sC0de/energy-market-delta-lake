If  the  User  identifies  that  the  Special  Read  is  no  longer  required,  the  User  will  forward  a SpecialReadRequest transaction to the Network Operator with the actionType set to 'Cancel' to identify that this is a cancellation.
If the Network Operator has received a cancellation notice from the User, the Network Operator will cancel the Special Read and forward a SpecialReadResponse to the User to confirm closure.
The response will contain a responseType set to "Closure" to identify that the Special Read is closed.
