This transaction is an ad-hoc type advice used by the Network Operator to notify the User of any planned meter changes it expects to undertake as a result of the Network Operator's meter time expired  program change. The User may use this information to advise customers and/or call centres of any such changes. It is initiated by the Network Operator and is passed to the User.
This transaction is not used in SA. For meter exchanges, please refer to T330.
