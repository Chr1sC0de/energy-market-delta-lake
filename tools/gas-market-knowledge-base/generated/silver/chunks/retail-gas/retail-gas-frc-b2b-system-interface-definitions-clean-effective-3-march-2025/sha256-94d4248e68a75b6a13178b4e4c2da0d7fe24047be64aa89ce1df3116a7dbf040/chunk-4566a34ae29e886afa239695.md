Route and Site Information transactions are initiated by both Users and Network Operators to maintain alignment of data relating to Customer Sites and the Meter Reading Schedule.
The following table shows the Route and Site Information group of aseXML transactions and the corresponding transactions from the  Table of Transactions.
AmendMeterRouteDetails, Table of Transactions.Ref No = 66. AmendMeterRouteDetails, Table of Transactions.Transaction Type = Meter Site Access Information Change from User. AmendMeterRouteDetails, Table of Transactions.Ref No = 67. AmendMeterRouteDetails, Table of Transactions.Transaction Type = Meter Site Access Information Change from Network Operator. AmendMeterRouteDetails, Table of Transactions.Ref No = 68. AmendMeterRouteDetails, Table of Transactions.Transaction Type = Supply Point Information. AmendMeterRouteDetails, Table of Transactions.Ref No = 69. AmendMeterRouteDetails, Table of Transactions.Transaction Type = Address Information Change from Network Operator. AmendMeterRouteDetails, Table of Transactions.Ref No = 75. AmendMeterRouteDetails, Table of Transactions.Transaction Type = Meter Reading Route Change
These business transactions will be mapped to the new 'SITE' Transaction Group in aseXML.
