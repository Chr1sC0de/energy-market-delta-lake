The  Network  Operator  must  make  available  in  an  electronic  form  a  remotely  accessed street/suburb combination listing for Users. It is used by the User to assist with MIRN Discovery requests (eg:  the  street  name  the  Network  Operator  uses  in  its  database).      The  date  of  file creation is to be taken as the date on which the data was updated.
The data is prepared by the Network Operator and can either be download from the Network Operator's website or e -mailed, if requested.
