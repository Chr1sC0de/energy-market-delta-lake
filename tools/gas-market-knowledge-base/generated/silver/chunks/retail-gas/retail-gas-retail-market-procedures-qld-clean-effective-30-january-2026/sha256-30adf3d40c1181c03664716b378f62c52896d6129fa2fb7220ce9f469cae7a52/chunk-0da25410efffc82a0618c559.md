Where:
- (a) a User delivers a transfer request in relation to a distribution supply point to AEMO ;
- (b) a transfer  request in  relation  to  that distribution  supply  point has  already  been delivered to AEMO (whether by that or any other User ); and
- (c) the transfer request referred to in clause 4.1.5(b) (the existing transfer request ) is being processed by AEMO but the User who delivered the existing transfer request to AEMO has not yet been registered in the AEMO meter register as the FRO for the distribution supply point to which the existing transfer request relates,
then AEMO must  not  process the transfer request referred to  in  clause  4.1.5(a)  and must,  by  midnight  on  the  first business  day following  the  day  on  which  that transfer request was  delivered  to AEMO ,  deliver  a  notice  stating  this  fact  to  the User who delivered that transfer request .
