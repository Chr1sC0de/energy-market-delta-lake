- (2) If at any time before the network operator has provided information to a user under clause  167(4),  a customer's explicit  informed  consent under  clause  166A(1) ceases to apply (for example because it is withdrawn), then the user must withdraw the  request  to  the  extent  that  the  request  relied  upon  the customer's explicit informed consent .
