- (1) A network operator has commissioned a delivery point when:
2. (a) a meter has been installed and commissioned at the delivery point ; and
3. (b) gas is able to flow at the delivery point ; and
4. (c) the meter reading data has been obtained for the delivery point .
- (2) Upon first commissioning a delivery point , a network operator must:
6. (a) promptly notify the user that the installation of the meter at the delivery point is complete and notify the user of:
7. (i) the MIRN standing data ; and
8. (ii) the meter standing data assigned to the delivery point ; and
AEMO
- (iii) for a basic-metered delivery point only, the metering data in accordance with clause 161 as if the user were an incoming user ; and
- (b) promptly lodge a new connection confirmation notice with AEMO.
