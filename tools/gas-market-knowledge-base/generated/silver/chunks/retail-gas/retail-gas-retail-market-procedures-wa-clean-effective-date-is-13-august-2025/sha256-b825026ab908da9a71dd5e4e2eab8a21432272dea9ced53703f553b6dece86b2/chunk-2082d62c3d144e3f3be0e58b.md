- (1) Subject to clause 167(2), a user may request a network operator to provide it with either or both of:
2. (a) historical metering data ; or
3. (b) historical meter reading data ,
for one or more of the user's delivery points for a period specified in the request.
- (2) By lodging a request under clause 167(1), the user represents and warrants to the network operator that either:
2. (a) the requested data relates only to a period for which the user was the current user ; or
3. (b) that the user has complied with clause 166A(1).
{Note:  Under clause 376A(2), a breach of this warranty will expose the user to liability for more than just direct damage.}
