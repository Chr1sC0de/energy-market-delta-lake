- (1) If a current user becomes aware of an error or inaccuracy in an item of the AEMO standing data as a result of:
2. (a) lodging an incorrect transfer request with AEMO, then the current user must as soon as practicable and in any event within 10 business days notify the previous user of this fact.  The previous user must as soon as practicable and in any event within 10 business days investigate the error or inaccuracy.
3. (i) There is no clause 32(1)(a)(i)
4. (ii) There is no clause 32(1)(a)(ii)
5. (iii) There is no clause 32(1)(a)(iii)
6. (iv) There is no clause 32(1)(a)(iv)
7. (b) the network operator having lodged an incorrect delivery point transaction with AEMO in respect of a new connection confirmation notice or permanent removal confirmation notice - the current user must as soon as practicable
and in any event within 10 business days notify the network operator of this fact.
