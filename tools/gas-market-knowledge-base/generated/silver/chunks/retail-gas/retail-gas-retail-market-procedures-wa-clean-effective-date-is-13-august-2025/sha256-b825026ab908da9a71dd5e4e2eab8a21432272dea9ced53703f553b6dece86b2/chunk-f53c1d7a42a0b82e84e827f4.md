- (1) A network operator :
- (a) may permanently  remove a delivery  point when  required  to,  or  if  not prevented, by law or a contract other than these procedures ; and
- (b) must (subject to law ) permanently remove a delivery point if required to under this clause 125.
- (2) Subject  to  clause  125(3),  a user may at  any  time  lodge  a permanent removal request with the network operator .
- (3) A user must not lodge a permanent removal request more than 20 business days before the date on which the user requires the delivery point to be permanently removed .
- (4) A permanent removal request :
- (a) must specify at least the following information:
- (i) the MIRN ; and
- (ii) the user's GBO identification ; and
- (iii) the earliest date that the delivery point can be permanently removed ; and
- (b) if the user chooses ⎯ may specify the service order reference .
- (5) A permanent removal request is valid only if:
- (a) the MIRN status is commissioned or decommissioned ; and
- (b) it relates to a delivery point in the network operator's sub-network ; and
