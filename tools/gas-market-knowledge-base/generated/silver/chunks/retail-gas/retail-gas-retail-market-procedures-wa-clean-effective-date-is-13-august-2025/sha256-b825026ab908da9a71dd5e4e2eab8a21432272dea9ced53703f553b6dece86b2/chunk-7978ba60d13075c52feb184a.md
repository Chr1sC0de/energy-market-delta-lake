- (1) A previous user may withdraw an error correction notice in respect of a transfer request at  any  time  before AEMO completes  the error  correction  notice under clause 46, by lodging an error correction withdrawal notice to AEMO.
- (2) A provision of these procedures permitting or requiring AEMO to cancel an error correction transaction does not limit the generality of clause 43(1).
- (3) An error correction  withdrawal notice must  specify at least the following information:
4. (a) details of the error correction notice to which the error correction withdrawal notice relates; and
5. (b) the GBO identification of the previous user lodging the notice .
- (4) An error correction withdrawal notice is valid only if it corresponds to an open error correction notice previously lodged by the previous user who has an active GBO identification .
