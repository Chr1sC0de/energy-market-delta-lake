Should a file with the same filename be submitted more than once, a file with the same name but a .DUP (for duplicate) will be placed in the participant's Outbox. The content of the .DUP file will take the form of an acknowledgement with a header row and a single data row with event code 5800 (duplicate zip filename). This will prevent the original copies of the .ACK file from being overwritten.
