To be consistent with the RMP this section refers to the previous user, however, for the purposes of the correction of transfer a previous user can be considered to be the same as the incoming user (as per a transfer process).
This is a notification to the current user, previous user and network operator to identify that an error correction has been cancelled. This would be because there was an objection, but no objection withdrawal received within the required period.
