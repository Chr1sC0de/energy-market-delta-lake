- (a) By 2.00 pm on nomination day -2, a Network Operator must notify a User of the User 's prior imbalance account in each network section for nomination day -4.
- (b) By 4.00 pm on nomination day -2, a User must notify the following details to the Network Operator for each network receipt point at which the User intends to receive gas :
- (i) for the nomination day , the quantity of gas the User seeks to receive in total and for each shipper , not including the User 's reconciliation adjustment amount .
