- (a) A User who delivers a transfer request to AEMO :
- (i) may deliver a notice withdrawing that transfer request (a transfer withdrawal notice ) to AEMO at any time before (but not after) the registration of that User in AEMO metering database as the FRO for the delivery point to which the transfer request relates; and
- (ii) must immediately deliver a transfer withdrawal notice to AEMO where it ceases to be a User at any time before the registration of that User in the AEMO metering database as the FRO for the delivery point to which the transfer request relates.
- (b) A transfer withdrawal notice must include the MIRN for the delivery point to which the transfer request relates.
