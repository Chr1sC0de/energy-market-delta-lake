If a User fails to notify the Network Operator of its forecast requirement and/or confirmed nomination , the Network Operator must determine the User 's confirmed nomination for a network receipt point based on:
- (a) the User 's forecast requirement adjusted for the User 's reconciliation adjustment amount for the same day in the prior week; or
- (b) the User 's forecast requirement , adjusted for the User 's reconciliation adjustment amount for the same day in the two weeks prior where the same day in the prior week is a public holiday in New South Wales.
