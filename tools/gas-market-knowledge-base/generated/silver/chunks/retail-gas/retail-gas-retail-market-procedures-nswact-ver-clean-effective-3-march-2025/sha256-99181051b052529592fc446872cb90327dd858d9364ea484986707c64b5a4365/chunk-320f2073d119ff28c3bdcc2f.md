- (a) Each User who is the FRO for a delivery point in a Network Operator's network must use its reasonable endeavours to provide that Network Operator with details of any changes to any site access information in relation to the meter for that delivery point by 5.00 pm on the next business day after the day on which the User receives those details. Those details must be identified to the Network Operator by reference to the MIRN for the delivery point .
- (b) Each Network Operator must use its reasonable endeavours to provide each User who is the FRO for a delivery point in its network with details of any changes to any site access information in relation to the meter for that delivery point by 5.00 pm on the next business day after the day on which the Network Operator receives those details. Those details must be identified to the User by reference to the MIRN for the delivery point .
