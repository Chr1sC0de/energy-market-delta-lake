- (i) The Network Operator may use Types 1, 2, 3, 4, 5 or 6 techniques in accordance with section A4.2(e), when metering data is required to be substituted.
- (ii) The Network Operator may use Types 1, 2, or 3 techniques in accordance with section A4.2(e), when metering data is required to be estimated.
- (iii) The Network Operator may use Type 6 where there is an error in the meter pressure and/or gas temperature.
- (iv) The Network Operator must notify the User where a substituted meter reading or estimated meter reading is used.
- (v) The Network Operator must ensure that for all Types, a substituted meter reading or estimated meter reading is based on an actual meter reading ,
