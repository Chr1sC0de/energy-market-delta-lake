- (a) On nomination day +1, AEMO must calculate the net section load for each network section for the nomination day in accordance with the following formula:
NSL = TDQ - TDM - UAG - CLP
If NSL < 0 then set NSL = 0
Where:
NSL = net section load for nomination day (in MJ );
TDQ = total quantity of gas injected (in MJ ) at all network receipt point s within the network section for the nomination day ;
TDM = total quantity of gas withdrawn (in MJ ) from all daily metered delivery points within the network section for the nomination day ;
UAG = unaccounted for gas (in MJ ) for the network section for the nomination day ; and
CLP = change in linepack (in MJ ) for the network section for the nomination day .
