If a Network Operator identifies a class of basic meter s that is required to be replaced on the basis of having reached the life expectancy of that class of meter the Network Operator must:
- (a) notify each FRO for a delivery point to which a meter which is a member of that class relates of its intention to replace all meters in that class;
- (b) ensure that the meters to be replaced will be replaced by meters which comply with any applicable laws; and
- (c) remove the meters requiring replacement and install the replacement meters in accordance with clauses 4.2 and 4.1 (as if the User had requested the installation of the replacement meter ) respectively.
