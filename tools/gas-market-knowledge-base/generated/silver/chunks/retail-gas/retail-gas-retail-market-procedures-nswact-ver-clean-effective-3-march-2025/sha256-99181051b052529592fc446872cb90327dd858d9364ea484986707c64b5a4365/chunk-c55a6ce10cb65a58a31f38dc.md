- (a) By 12.00 noon on nomination day +2, AEMO must notify the Network Operator of the following information for each User for each network section for the nomination day :
- (i) total estimated withdrawal ; and
- (ii) the User 's corrected reconciliation adjustment amount .
- (b) Where adjustments have been made as described in clause 8.9.17(d) then by 12.00 noon on nomination day +2, AEMO must notify the Network Operator of each User 's total estimated withdrawals for each network section for each nomination day .
