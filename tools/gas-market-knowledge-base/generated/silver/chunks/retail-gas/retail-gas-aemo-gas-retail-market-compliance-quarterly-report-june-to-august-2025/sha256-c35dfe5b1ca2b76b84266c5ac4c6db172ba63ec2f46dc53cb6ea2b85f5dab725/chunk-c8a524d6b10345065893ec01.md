RMP, by no later than 4.5 hours after the beginning of each gas day, AEMO must determine the STTM distribution system allocations for each User for each STTM withdrawal zone and distribution region for the previous gas day. On 8 August, the provision of the daily STTM distribution system allocations (the network allocation daily (NAD) file) for gas day 7 August for the QLD gas retail market to the STTM system was delayed by 1 minute. This caused a non-compliance with QLD RMP relating to timely provision of NAD file to the STTM system. Cause On 8 August, AEMO received an alert and identified the scheduling service failed to import interval energy and generate QLD NAD file due to the active database node being unresponsive. Actions On 8 August, the active database node was failed over to another node and the database server node was rebooted. The QLD NAD file generated and delivered to the STTM system successfully at 10.31AM (AEST). AEMO provided information to the vendor for review and investigation of the database server failure.. 8 August 2025, Impact = Immaterial A EMO's non -compliance with clause 6.10.2(a) of QLD
