In order to manage the 'transfer' and customer set up following a RoLR event, all non-Declared Host Retailers are to provide to the Retailer of Last Resort a list of MIRNS and associated details for which they are the FRO at the time of the RoLR event.
The trigger for this data delivery will be AEMO issues a Suspension Notice to the FRB.
The method of data delivery will be via an agreed method between the parties and not via the FRC Hub to reduce delays in data delivery.
The following file naming convention is to be used:
VICGAS_CUSTOMERSITEDETAILSFRB_OriginatorID_RecipientID_CCYYMMDDHHmmSS
