The following fields are provided in the report.
TransactionStartDate, Description = Transaction start date. TransactionStartDate, Data type = Date. TransactionStartDate, Examples = 2022-08-01. TransactionEndDate, Description = Transaction end date. TransactionEndDate, Data type = Date. TransactionEndDate, Examples = 2022-08-31. VolWeightPrice, Description = The volume weighted price for the reporting period. VolWeightPrice, Data type = Number(18,8). VolWeightPrice, Examples = 10.45. Volume, Description = The total volume of the transactions for the reporting period. Volume, Data type = Number(10,3). Volume, Examples = 2.333. SupplyStartDate, Description = The earliest start date of all transactions captured in the reporting period. SupplyStartDate, Data type = Datetime. SupplyStartDate, Examples = 2022-05-01. SupplyEndDate, Description = The latest end date of all transactions captured in the reporting period. SupplyEndDate, Data type = Datetime. SupplyEndDate, Examples = 2022-05-31
