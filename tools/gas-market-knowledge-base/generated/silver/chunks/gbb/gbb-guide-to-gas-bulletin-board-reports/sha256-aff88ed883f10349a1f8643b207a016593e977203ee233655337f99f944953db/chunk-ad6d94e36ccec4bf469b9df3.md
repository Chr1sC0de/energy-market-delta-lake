Transaction report name, 1 = GASBB_ACTUAL_FLOW_STORAGE / GASBB_ACTUAL_FLOW _STORAGE_LAST_31. Purpose, 1 = The report shows Daily Production, Flow and Storage data aggregated by Facility Id for an outlook period.. Update interval, 1 = Daily. Production Frequency, 1 = GASBB_ACTUAL_FLOW _STORAGE is updated daily / GASBB_ACTUAL_FLOW _STORAGE_LAST_31 is updated within 30 minutes of receiving new data.. Report Period, 1 = GASBB_ACTUAL_FLOW _STORAGE report shows historic records back to Sep 2018. GASBB_ACTUAL_FLOW _STORAGE_LAST_31 shows records from the last 31 days.
