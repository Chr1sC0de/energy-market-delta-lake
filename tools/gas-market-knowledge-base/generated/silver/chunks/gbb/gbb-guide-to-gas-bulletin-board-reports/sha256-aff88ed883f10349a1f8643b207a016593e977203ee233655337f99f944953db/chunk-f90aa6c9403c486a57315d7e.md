The following fields are provided in the report.
FacilityName, Description = Facility name associated with the Facility Id.. FacilityName, Data type = varchar(100). FacilityName, Example / Allowed values = APLNG Pipeline. FacilityId, Description = A unique AEMO defined Facility identifier.. FacilityId, Data type = Int. FacilityId, Example / Allowed values = 520345. FacilityType, Description = Facility type associated with the Facility Id.. FacilityType, Data type = varchar(40). FacilityType, Example / Allowed values = PIPE; PROD; STOR. CapacityType, Description = Capacity type can be either: • Storage: Holding capacity in storage, or • MDQ: Daily maximum firm capacity (name plate) under the expected operating conditions adjusted any facility that is 'mothballed', decommissioned or down-rated and / or cannot be recalled within 1 week, planned maintenance excepted. Reflects any. CapacityType, Data type = varchar(20). CapacityType, Example / Allowed values = STORAGE; MDQ
