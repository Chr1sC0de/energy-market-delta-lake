Transaction report name, 1 = GASBB_SHORT_TERM_CAPACITY_OUTLOOK / GASBB_SHORT_TERM_CAPACITY_OUTLOOK_FUTURE. Purpose, 1 = This report displays the expected daily capacity of a BB facility for the next seven days. Update Interval, 1 = GASBB_SHORT_TERM_CAPACITY_OUTLOOK is updated daily. GASBB_SHORT_TERM_CAPACITY_OUTLOOK_FUTURE is typically updated within 30 minutes of receiving new data.. Report period, 1 = GASBB_SHORT_TERM_CAPACITY_OUTLOOK contains historic outlooks. GASBB_SHORT_TERM_CAPACITY_OUTLOOK_FUTURE contains data in the seven day outlook window.
