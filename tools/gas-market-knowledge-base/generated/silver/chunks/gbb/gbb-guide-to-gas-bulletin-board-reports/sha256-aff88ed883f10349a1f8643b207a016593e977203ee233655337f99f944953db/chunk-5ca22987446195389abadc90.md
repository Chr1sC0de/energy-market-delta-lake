FacilityType, Description = Facility type associated with the facility id. FacilityType, Data type = Varchar(40). FacilityType, Example = PIPE,PROD,STOR,COMPRESSOR. ReceiptLocationId, Description = The Connection Point Id that best represents the receipt location associated with a pipeline's nameplate capacity flow direction.
