Details and examples of transaction data in CSV format are provided in transaction specifications. See Appendix B.
A comma in a free text field (such as Capacity Description and Description) is treated as a value separator in the validation process and results in validation errors. To prevent this validation error, exclude commas in the description, or enclose the description with commas in double quotes "".
