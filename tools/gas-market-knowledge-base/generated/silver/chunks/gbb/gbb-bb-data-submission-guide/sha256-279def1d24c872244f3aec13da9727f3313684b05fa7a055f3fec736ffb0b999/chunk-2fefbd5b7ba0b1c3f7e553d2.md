- Data only to be submitted by connection point for BB pipeline s or BB blended distribution systems if only certain parts of the BB facility transport a gas blend .
- For each connection point in a submission, check the connection point's 'Flow Direction' as defined in the Detailed Information Facility in the BB.
- Submitted connection points must be registered against the Facility Id of the BB pipeline and BB blended distribution system during the connection point registration process.
- Submissions must only contain BB pipelines and BB blended distribution systems for which the BB reporting entity is registered.
- All quantities and percentages must be submitted to three decimal places.
Data submission examples are listed in Appendix B for CSV file submissions. Visit AEMO developer portal for HTTP POST request submission examples.
