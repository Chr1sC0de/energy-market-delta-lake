- DevelopedReserve3P must be >= zero.

- UndevelopedReserve3P must be >= zero.

- ProvedAreaExtensionReserve2P must be >= zero.
- UpwardRevisionFrom3PReserveTo2P must be >= zero.
- DownwardRevisionFrom2PReserveTo3P must be <= zero.
