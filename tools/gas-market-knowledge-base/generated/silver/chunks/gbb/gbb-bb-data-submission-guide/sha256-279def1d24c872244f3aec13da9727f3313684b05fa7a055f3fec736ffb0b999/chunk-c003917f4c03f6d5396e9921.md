25.2 (if the value is 25.200). Flow Direction, Date field name = FlowDirection. Flow Direction, Description = Gas Flow Direction can be: Receipt: The flow of gas into the BB storage facility or LNG export facility Delivery: The flow of gas out of the BB storage facility or LNG import facility Processed: The flow direction type only used for capacities. For LNG export facilities , it represents the amount of gas that can be processed to a liquefied state on a gas day. For LNG import facilities , it represents the amount. Flow Direction, Mandatory = Conditional This field is mandatory for BB storage (MDQ Capacity Type), LNG Export and LNG Import facilities. Otherwise leave this blank..
