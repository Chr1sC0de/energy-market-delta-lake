- GasDate must conform to the date format YYYY-MM-DD.
- Submissions must only contain Facility Ids registered to the Company Id.
- Negative values are not accepted for the receipt or delivery Nomination Quantity.
- Users will be able to submit nomination and forecast flow data for any future gas date.
- Connection Point IDs are only accepted where facility type is Pipeline.
