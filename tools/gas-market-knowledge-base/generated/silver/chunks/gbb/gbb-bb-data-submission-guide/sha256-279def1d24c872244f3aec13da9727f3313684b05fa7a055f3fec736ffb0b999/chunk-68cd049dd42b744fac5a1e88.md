- Outlook Quantity must be submitted in TJs accurate to three decimal places.
- BB pipelines are required to submit capacities for each direction in which gas can be transported on the pipeline. An Outlook Quantity must be submitted with a Capacity Description and the Delivery and Receipt Points. Note that for complex pipeline facilities that involve more than two directions of flow, more than two capacities may be required.
- BB storage facilities are required to report capacity for receipts into, and deliveries from, the BB storage facility as well as the quantity of gas that can be held in storage.
- LNG Import facilities are required to report capacity for regassification by the LNG Import facility as well as the capacity for storage in the facility.
- BB Compression facilities and BB Production facilities must report the capacity available in the facility.
- Where a facility's capacity is reduced to zero, a zero value must be submitted.
- Submissions must only contain BB pipelines, BB storage facilities, BB blend processing facilities, BB Production facilities, LNG import or BB compression facilities for which the BB Reporting Entity is registered.
Data submission examples are listed in Appendix B for CSV file submissions. Visit AEMO developer portal for HTTP POST request submission examples.
