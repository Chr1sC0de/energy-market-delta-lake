Identifiers for linepack zones (LinepackZoneId) used in transactions and reports subscribe to the following format:
1, Description = 3, 4 or 5 character reference to identify the pipeline. 1, Values = SEP, SESA, SGP, etc. 2, Description = Specifies linepack zone. 2, Values = -LP-. 3, Description = Zone number on that pipeline. 3, Values = 01 to 99
Linepack zone identifiers have the following characteristics:
- Identifiers are defined and allocated by AEMO in accordance with the East Coast Gas System Procedures .
- A Part 27 relevant entity may report on multiple linepack zone identifiers.
For example, Identifier SESA-LP-01 relates to a linepack zone on the South East South Australia pipeline with unique identifier of '01' for that pipeline.
