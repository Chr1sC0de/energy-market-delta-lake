- GasDate must conform to the date format YYYY-MM-DD.
- Submissions must only contain Facility Ids registered to the Company Id.
- Negative values are not accepted for the Outlook Quantity.
- BB pipeline and BB compression facilities must submit a Capacity Description.
- All facilities must include Description in their submissions (unless submitting a file to deactivate all future MTCO's)
- Recall time must be greater than zero.
- For CSV file submissions, Description and Capacity Description with commas must be enclosed in double quotes, or exclude any commas.
