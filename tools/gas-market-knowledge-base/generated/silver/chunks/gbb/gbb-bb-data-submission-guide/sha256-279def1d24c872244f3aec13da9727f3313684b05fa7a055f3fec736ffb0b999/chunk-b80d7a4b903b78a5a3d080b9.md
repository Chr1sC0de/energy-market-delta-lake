- Effective Date must conform to the date format YYYY-MM-DD.
- Submissions must only contain Connection Point Ids connected to a BB pipeline or BB compression facility registered to the CompanyId.
- A Capacity Quantity value of zero must be submitted if there is no ability to flow gas.
- For CSV file submissions, Description and Capacity Description with commas must be enclosed in double quotes or exclude any commas.
