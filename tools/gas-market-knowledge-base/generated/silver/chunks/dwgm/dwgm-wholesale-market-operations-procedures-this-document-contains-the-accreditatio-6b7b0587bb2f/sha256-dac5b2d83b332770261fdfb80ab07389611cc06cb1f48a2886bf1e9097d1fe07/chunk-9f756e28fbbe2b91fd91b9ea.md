A NFTC allows multiple system injection points and system withdrawal points at a common location to be combined so that the net aggregate flow is constrained to reflect the physical DTS capacity (e.g. pipeline capacity).
The NFTC parameters are:
Table 12 NFTC parameters

maximum hourly net injection quantities, Description = the maximum net amount of gas that can be injected in an hour;. maximum hourly net withdrawal quantities, Description = the maximum net amount of gas that can be withdrawn in an hour;. minimum hourly net injection quantities, Description = the minimum net amount of gas that can be injected in an hour;. minimum hourly net withdrawal quantities, Description = the minimum net amount of gas that can be withdrawn in an hour;. maximum daily net injection quantities, Description = the maximum net amount of gas that can be injected over a day. maximum daily net withdrawal quantities, Description = the maximum net amount of gas that can be withdrawn over a day.. minimum daily net injection quantities, Description = the minimum net amount of gas that can be injected over a day. minimum daily net withdrawal quantities, Description = the minimum net amount of gas that can be withdrawn over a day.
