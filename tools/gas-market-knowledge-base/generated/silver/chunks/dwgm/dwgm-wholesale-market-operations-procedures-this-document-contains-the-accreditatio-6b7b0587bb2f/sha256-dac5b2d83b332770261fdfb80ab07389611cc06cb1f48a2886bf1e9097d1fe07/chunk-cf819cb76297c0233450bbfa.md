The MCE reference data includes Node configuration, withdrawal zones, system linepack, and VoLL price.  This data is necessary for the MCE to be able to produce Feasible Operating Schedules. AEMO will:
- (a) publish the details of Node configuration, system linepack and withdrawal zones on the MIBB;
- (b) apply an appropriate change management process to make any changes to MCE reference data; and
- (c) notify Market Participants by SWN of any significant changes made to MCE reference data.
