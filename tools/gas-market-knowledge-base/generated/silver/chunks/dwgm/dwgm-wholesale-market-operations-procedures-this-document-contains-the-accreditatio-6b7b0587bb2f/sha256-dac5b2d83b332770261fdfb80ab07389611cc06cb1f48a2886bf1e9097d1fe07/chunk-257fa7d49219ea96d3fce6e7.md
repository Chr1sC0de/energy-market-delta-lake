Table 10 SDPC constraint parameters

Hourly ramp down rate, Description = The maximum hourly ramp down rate for the meter.. Hourly ramp up rate, Description = The maximum hourly ramp up rate for the meter.. Minimum hourly quantity, Description = The minimum hourly injection/withdrawal quantity. This value is 0/GJ/h by default unless set to a different value by AEMO.. Maximum hourly quantity, Description = The maximum hourly injection/withdrawal quantity. Hourly response time, Description = The amount of time in hours for the facility to respond to the constraint.. Flexible response, Description = Indicates that the scheduled rate can change during a gas day.. Point expiration time, Description = The time at which the constraint expires. By default, AEMO sets this value to 5AM unless set to a different value in consultation with the DWGMfacility operator .. Daily minimum quantity, Description = The minimum daily injection/withdrawal quantity. This value is 0/GJ/h by default unless set to a different value by AEMO.. Daily maximum quantity, Description = The maximum hourly injection/withdrawal quantity
