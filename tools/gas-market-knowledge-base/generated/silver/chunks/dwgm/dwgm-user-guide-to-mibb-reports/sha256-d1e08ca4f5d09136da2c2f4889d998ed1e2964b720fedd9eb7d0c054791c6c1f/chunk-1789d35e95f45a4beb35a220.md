This report is generated at Provisional, Final, and Revised settlements for distribution businesses and retailers.
Each report issued for a gas day can expect to have fewer estimated basic meters over time.
The report groups the counts by the gas date and logical mirn. The mirn is related to the distributor, financially responsible organisation, withdrawal zones.
This report is similar to VIC MIBB report INT473.
