Each report contains the:
- l auction identifier
- l date when auction was run
- l zone identifier
- l zone's name
- l zone's type
- l start and end period associated with the CC product
- l CC product period's name
- l cleared price per auction product
- l quantities won for each product
- l unallocated quantities for each product
- l date and time when report was produced
