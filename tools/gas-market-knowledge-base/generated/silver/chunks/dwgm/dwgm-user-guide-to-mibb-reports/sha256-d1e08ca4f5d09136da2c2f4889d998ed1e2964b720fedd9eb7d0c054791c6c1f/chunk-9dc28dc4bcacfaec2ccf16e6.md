This report is the same report as INT137. The only difference being that it is generated at a settlements run and reflects actual payments.
Please note that this report should only be used for settlement revision up until gas day: 31 Dec 2022. There will be new Uplift report (INT290b) from gas day: 1 Jan 2023.
