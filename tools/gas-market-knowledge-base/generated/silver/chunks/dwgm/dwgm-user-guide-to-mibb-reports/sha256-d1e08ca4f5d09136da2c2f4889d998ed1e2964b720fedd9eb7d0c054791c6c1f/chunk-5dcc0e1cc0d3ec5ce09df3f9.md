The INT801 file is only generated where the level of errors within the file is under 15%. When the file error rate is above 15%, the retailer receives an email notification only.
The 15% error rate is configurable at the discretion of AEMO and may change. In such instances, AEMO will provide appropriate notice.
The file is delivered within four business hours of the completion of the file validation process.
The report contains a list of all the validation errors that were detected while validating the T900 (Customer and Site Details Monthly) file.
