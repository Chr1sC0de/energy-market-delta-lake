The report for each allocation agent contains details of registered Market participants for a biddable injection or withdrawal point only.
This report contains details of the latest approved operational schedules for:
- l the current gas day
- l 1 gas day in the future and
- l 2 gas days in the future (if an approved 2-day ahead schedule exists at the time the report is run).
For the purposes of this report:
- l 'type' always contains the value 'a'
- l 'flag' always contains the value 'OS' and refers to an Operational Schedule.
