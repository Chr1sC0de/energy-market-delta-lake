This report is generated daily. Each report displays the success or otherwise of each hourly download of gas composition details for each meter controlled by a Declared Transmission System Service Provider (DTSSP) over the previous day.
Each row in the report provides:
- l the result of transfers for each hour of the specified gas day for a specified meter
- l a count of the number of successful transfers over the gas day
- l a count of the number of failed transfers over the gas day
- l a count of the incomplete transfers over the gas day.
Where transfer of data was successful in an hour, the report will contain a null.
Where the download of hourly gas composition data to the meter failed, the report will contain an 'F'.
Where the download of hourly gas composition data to the meter was incomplete, the report will contain an 'N'.
