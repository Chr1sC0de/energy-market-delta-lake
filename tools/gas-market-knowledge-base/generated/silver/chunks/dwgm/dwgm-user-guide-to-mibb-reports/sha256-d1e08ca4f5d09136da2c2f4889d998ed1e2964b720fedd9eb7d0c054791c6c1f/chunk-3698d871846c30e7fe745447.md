This report provides confirmation of the quantities that the specified Market participant is scheduled to flow over the gas day, split into hourly intervals from the start of the gas day. The report is generated on each approved operating schedule. This report is configurable and is aimed to deliver only the schedules required for the purpose of gas allocation.
