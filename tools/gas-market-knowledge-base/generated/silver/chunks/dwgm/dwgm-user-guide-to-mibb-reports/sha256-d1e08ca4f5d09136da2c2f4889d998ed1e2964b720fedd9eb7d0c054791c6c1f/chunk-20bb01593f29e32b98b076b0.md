This report provides the daily net system load (NSL) for each distribution area for a non-DTS (declared transmission system) network for the past 3 years. This report may be used as a reference for settlement information.
Section 2.8.4 of the Victorian Retail Market Procedures AEMO's obligation to publish the NSL and Attachment 6 of the Victorian Retail Market Procedures set out how AEMO calculates the NSL
