This report is a comma separated values (csv) file that contains each Part 27 relevant entity's active ECGS Responsible Person contacts. The ECGS Respsonible Person contact receives ECGS notices from AEMO. This report is published to the:
- l Market Information Bulletin Board (MIBB) public folder
- l NEMWEB folder: https://www.nemweb.com.au/REPORTS/CURRENT/ECGS/
