Each report contains the:
- l gas date
- l schedule interval (indicating 1 to 5 when the deviation occurred, where 1 refers to 6:00 AM to 10:00 AM, 2 will relate to 10:00 AM to 2:00 PM, and so forth)
- l transmission identifier for the schedule
- l scheduled injections in gigajoules
- l scheduled withdrawals in gigajoules
- l price for the scheduling horizons
- l Administered Price (the value in the admin price field is null when no admin prices applies and when there has been an admin price, it will be displayed and the price_value will show the last approved schedule price)
- l Actual metered withdrawals in gigajoules
- l Actual metered injections in gigajoules
