Each report contains the:
- l network name
- l BMP (basic meter profile) run id
- l mirn
- l gas date
- l trading interval (where 1= 6:00 AM to 7:00 AM, 2 = 7:00 AM to 8:00 AM, and so forth until 24 intervals)
- l hourly energy in gigajoules
- l unaccounted for gas on an hourly energy basis in gigajoules
- l quality description
- l extract type
- l date and time when the report was produced
