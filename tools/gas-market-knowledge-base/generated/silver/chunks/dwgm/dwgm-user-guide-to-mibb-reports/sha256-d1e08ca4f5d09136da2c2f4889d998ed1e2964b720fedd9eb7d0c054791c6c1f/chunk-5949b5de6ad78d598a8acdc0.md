This report provides the detailed breakdown of the Market participant's tie-breaking rights. It is generated on the approval of an operating schedule for the current gas day (D), D+1 or D+2.
Market participants should review this report each time an operating schedule is published.
Note that some Market participants receive an empty report (such as, a report with the column names only) if tie breaking right values are not relevant to them.
