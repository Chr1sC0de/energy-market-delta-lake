This report is generated each time AEMO generates and saves a demand forecast. Each report provides details of all the demand forecasts created up to the report generation time on the current gas day.
A given report may contain demand forecasts that are for:
- l the current day
- l 1 day ahead
- l 2 days ahead
- l 3 days ahead
By comparing the forecast_date with the current_date, users will be able to determine the type of forecast for a particular forecast (version_id) (i.e. whether it is for the current day or a day ahead schedule)
Each row in a report provides the forecast demand and the AEMO override (in GJ) for the specified hour of the gas day of the specified demand forecast. Therefore, there will be 24 rows in the report for each demand forecast.
