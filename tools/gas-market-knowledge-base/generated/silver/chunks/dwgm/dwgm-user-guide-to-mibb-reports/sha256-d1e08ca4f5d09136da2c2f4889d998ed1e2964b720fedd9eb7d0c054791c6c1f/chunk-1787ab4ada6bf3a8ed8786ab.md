Contains current metering data by Injection and Withdrawal MIRN.
This report is only for specified Allocation Agent Meters .
This is a Participant specific Report , produced on validation of CTM data. The date range contained within the report will reflect the latest reading dates process as part of D+3 energy processing
(i.e. the file will contain energy from the 1st of the (d-3) month to d-3 for Daily processing, and a complete month for Preliminary, Final or Revision processing).
