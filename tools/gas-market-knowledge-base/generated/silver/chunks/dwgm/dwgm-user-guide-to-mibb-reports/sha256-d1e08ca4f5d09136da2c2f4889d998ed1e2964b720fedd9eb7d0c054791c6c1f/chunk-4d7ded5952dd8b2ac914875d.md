Note the information in this report is derived from real-time gas pressure data and does not relate to the scheduled linepack values, as the scheduled linepack is a relative value.
It is also important to recognise that this report contains details about the system's physical linepack and is not the settlement linepack, which is a financial balancing concept not related in any way to the physical linepack in the system on each day.
Participants may use this report to make assumptions about the physical capabilities of the system when correlated with weather, type of day and other variables that impact on demand.
