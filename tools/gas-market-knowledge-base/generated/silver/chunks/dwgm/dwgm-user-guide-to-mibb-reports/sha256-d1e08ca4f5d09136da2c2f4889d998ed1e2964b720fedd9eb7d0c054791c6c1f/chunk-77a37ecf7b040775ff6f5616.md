Each report contains the:
- l gas date
- l weighted average daily price for imbalance and deviation (GST exclusive)
- l date and time when the report was produced
The report should contain one row representing each gas day in a month. Therefore in a month consisting of 30 days, the user can expect to see 30 rows of data.
