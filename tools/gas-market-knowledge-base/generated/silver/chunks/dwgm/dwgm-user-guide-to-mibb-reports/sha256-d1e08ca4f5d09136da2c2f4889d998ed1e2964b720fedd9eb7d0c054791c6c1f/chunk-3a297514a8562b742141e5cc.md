"Like Site": This rule allows data from a site with a similar load profile to be used to substitute the data. the three parameters (values) are:"MIRN" the MIRN of the "like site", "Reading Type" which is either "E" = Energy or "V" = Volume and "Factor (0.9=90%)" which is the proportionality between the like site and the one to be substituted.
"Default substitution rule": the parameter (value) is a fixed number to be substituted (in GJ for energy and kscm for volume).
