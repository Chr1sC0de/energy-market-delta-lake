MIBB report can be delivered as csv files or compressed into zip files. The file name includes the following elements, separated by an underscore:
- l int number
- l version
- l filename
- l participant ID
- l dynamic datestamp (yyyymmddhhmmss)
The following are examples of filenames:
Participant, 1 = int230_v1_basic_meter_count_6~20041116092632.csv. Public, 1 = int126_v2_FDS_data_rpt_1~20041214072353.csv. Zipped file, 1 = int254_v4_metering_data_monthly_234~20041209112730.zip
