This report provides daily total quantities and the percentage allocation to each Market participant. It compliments the INT124 report that provides hourly quantities.
For the purposes of this report:
- l record type is always a
- l offer type is always injec
See "INT108 - Schedule Run Log" on page 34 to determine the characteristics of each schedule(for example, the schedule start date and time, publish time and so on) associated with a specific transmission id (which is also known as schedule id).
