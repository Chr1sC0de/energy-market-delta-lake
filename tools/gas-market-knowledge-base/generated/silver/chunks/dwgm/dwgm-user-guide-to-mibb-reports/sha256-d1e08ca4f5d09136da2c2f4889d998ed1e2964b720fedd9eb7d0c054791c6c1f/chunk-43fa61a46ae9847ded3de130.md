Each report contains 7 days of data for the DTS Network specific to each participant.
Each report contains the:
- l mirn
- l gas date (refers to the date the record was current)
- l node information (the AEMO node used in the AEMO Market Clearing Engine)
- l withdrawal zone information (the zone to which the site load is allocated)
- l pricing zone information (in the current market there is only one price zone)
- l heating value information (identified the heating value zone to which the site is allocated)
- l injection withdrawal (identifies whether the site is injecting into the DTS or withdrawing from it)
- l location details
- l CTM meter number (AEMO's unique identifiers for transmission connected sites)
- l TUoS information (the TUoSzone to which the site is allocated)
- l uafg information (the loss factor applied to this site)
- l amdq and amdq credit information
- l name of financially responsible organisation, distributor and retailer
