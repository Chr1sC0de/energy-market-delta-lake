Each report contains aggregated consumption data for all gas days in the settlement period for each retailer.
The report will contain the following:
- l network name
- l statement version identifier
- l name of the financially responsible organisation, distributor and retailer
- l gas date
- l distributors name to identify the distribution region
- l withdrawal zone information
- l aggregated consumption in GJs
- l current date
