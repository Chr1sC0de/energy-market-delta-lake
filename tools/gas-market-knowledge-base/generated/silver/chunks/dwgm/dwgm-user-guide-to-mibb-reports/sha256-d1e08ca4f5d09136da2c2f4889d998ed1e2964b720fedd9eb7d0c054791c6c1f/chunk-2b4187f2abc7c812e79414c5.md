This report contains details for each of the schedules run for the previous 7 gas days. It includes both:
- l standard schedules (such as the schedules published at fixed times specified in the NGR); and
- l any ad hoc schedules
Information in the report can be linked to Participants demand forecasting activities. For example, see " INT126 - DFS Data" on page 50 and "INT153 - Demand Forecast" on page 61 for quantities and prices.
