Each report contains the:
- l gas date
- l mirn identifier
- l fro details
- l mirn checksum (used to validate that the correct mirn is assigned)
- l base load (the summer load for that site)
- l temperature sensitivity factor (the factor used along with the base load to estimate daily consumption)
- l post code
- l date and time when the report was produced
