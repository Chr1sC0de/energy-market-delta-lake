The report is generated for each T900 (Customer and Site Details Monthly) file provided by the retailer within 24 hours of the completion of the file validation process.
The report contains a list of all the validation errors that were detected while validating the T900 (Customer and Site Details Monthly) file.
The report contains validation errors for both DTS and non DTS MIRNs related to the retailer who submitted the T900 (Customer and Site Details Monthly) file.
