This report is produced after the approval of each schedule. The report has the actual price information of each operating schedule, as well as forecast process for the day ahead schedules published. This means that on any given gas day the report is published:
- l 5 times for Day+0 after each of the 5 market schedules which reflects actual price that applies for each scheduling horizon.
- l 3 times for Day+1 which includes the estimated BoD price.
- l Once for Day+2 which includes the estimated BoD price.
Participants can use this report to review the outcomes of the current day and to reflect on the options to adjust their positions for the coming days.
