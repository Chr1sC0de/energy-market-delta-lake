This report shows the deviation charges for each participant for each schedule in a gas day.
Participants may wish to use this report to view their deviation amount and deviation price.
Note as deviation utilise actual metered values the records in the report that have a statement version ID ='0' are subject to provisional meter data and may change at settlement.
