This report is generated daily. Each report displays the daily state HV for the previous 90 days (not including the current gas day).
Each row in the report provides the daily state HV for the specified gas_date.
Since daily state HV is calculated on the basis of hourly HV readings, the latest possible daily state HV available is for the previous full gas day. This means that daily state HV is always published 1 day in arrears.
