A single page schematic with colour coded symbols as outlined in Table 8 is required for each gas quality monitoring system . The schematic must show the source(s) of the gas, configuration of the site and location of all gas quality monitoring system related equipment (e.g. connected pipelines, storage or production facility(s), gas quality sample points (GC, sulphur, water, oxygen) odorant injection, flow meters, line valves, heaters and gas flow direction indication).
Table 8 Standard schematic symbols

, Description = C6+ GC Sample Point. , Description = C9+ GC Sample Point. , Description = Water Analyser Sample Point. , Description = Sulphur Analyser Sample Point. , Description = Oxygen Analyser Sample Point. , Description = Odorant Injection
, Description.Valve = Flow Meter. , Description.Valve = Heater
